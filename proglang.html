Chapter 6: Programming Languages


Data Types:

  • integer - represents a range of mathematical integers
  • float - represents a number that is not an integer
  • string - represents a sequence of characters
  • primitive data type - includes byt, short, int, long, float, double, boolean, char
  • data structure - an organization, management, and storage format that is usually chosen for efficient access to data

Python collection data structures:

  • list - stores multiple items in a single variable, and is written with square brackets '[]'
  • tuple - is ordered and unchangeable, and is written with round brackets '()'
  • set - is unordered, unchangeable, and unindexed, and is written with curly brackets '{}'
  • dictionary - is ordered, changeable, and does not allow duplicates, and is written with curly brackets '{}' and has keys and values

  • function - a block of code that only runs when it is called
  • local variable - variable declared within a function and can only be used within that function
  • global variable - vairable that is accessible throught the entire program
  • source program - a human readable program that is written by a programmer
  • object program - a program into which some other program is translated by an assembler or compiler
Term Definiton
machine language a computer programming language consisting of binary or hexadecimal instructions which a computer can respond to directly
debugging the process of identifying and removing errors from computer hardware or software
assembler a program used to convert instructions written in low-level symbolic code into machine code
assembly language low-level symbolic code converted by an assembler
translator a program that translates from one programming language to another
compiler a program that converts instructions into a machine-code or lower-level form so that they can be read and executed by a computer
interpreter a program that can analyze and execute a program line by line
cross-platform software a computer software that is designed to work in several computing platforms
natural language a language that has developed naturally in use
formal language a language that consists of words whose letters are taken from an alphabet and are well-formed according to a specific set of rules
machine independence a term applied to software that can run on a variety of different computers
high-level programming language a programming language with strong abstraction from the details of the computer
object-oriented programming (OOP) a programming paradigm that focuses on using objects as building blocks of software
scripting language a programming language that is used to manipulate, customize, and automate the facilities of an existing system
