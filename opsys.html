Operating System Terminology

Term Definition
operating system software that controls the overall operation of a computer
job queue a storage organization where jobs reside waiting for execution
FIFO stands for first-in-first-out. objects are removed from the queue in the order they were added
interactive processing feature that allows a program being executred to carry on a diologue with the user
time-sharing feature that allows operating systems to assist multiple users at once
multiprogramming process in which time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
application software consists of programs for performing tasks particular to the machine's utilization
utility software consists of programs for performing activities that are fundamental to computer installations.
graphical user interface (GUI) the interface that allows users to interact with electronic devices, such as computers, laptops, smartphones and tablets, through graphical elements.
shell older user interfaces that communicated with users through textual messages using a keyboard and monitor.
kernel internal part of an operating system that contains software components that perform very basic functions required by the computer installation.
device driver software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
file manager component of operating system's kernel that coordinates the use of the machine's mass storage facilities.
memory manager component of operating system's kernel that coordinates the machine's use of main memory
virtual memory large "fictional" memory space created by paging that allows a computer to run more efficiently
boot strapping performed by a computer each time it is turned on in order to start the operating systems
ROM memory that can be read but not altered
deadlock the condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
spooling technique of holding data for output at a later but more convenient time in order to grant several processes access to a common resource
super user (or administrator) User that has highly privileged access to the operation system
multi-factor authentication an electronic authentication method in which a user is granted access to a website or application only after successfully presenting two or more pieces of evidence (or factors) to an authentication mechanism.
